## 📝 Problem
Create a list of numbers: [1, 4, 7, 10].
Use a for loop to iterate through the list.
Inside the loop, print each number multiplied by 3.
## Expected Output 
3
12
21
30

---

## 💻 Solution
```python
numbers=[1,4,7,10]

for num in numbers:
    print(num*3)
