# 🧠 Exercise: Loop Through a List and Apply Transformation

## 📝 Problem
Work with lists and loops by iterating through a collection of numbers and applying a simple transformation.

## Requirements
- Create a list containing numeric values.
- Use a `for` loop to iterate through each element.
- Multiply each number by 3.
- Print the result of each operation.

## Expected Output
3  
12  
21  
30  

## Concepts Practiced
- Lists
- For loops
- Iteration
- Basic arithmetic operations

## Source
Exercise inspired by a Python course on Udemy.

---

## 💻 Solution
```python
numbers=[1,4,7,10]

for num in numbers:
    print(num*3)
