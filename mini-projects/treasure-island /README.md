# 🏝️ Treasure Island Game (Python)

A simple interactive text-based adventure game built with Python.  
The player must make the right decisions to find the treasure.

---

## 🎯 Objective

Guide the player through a series of choices.  
Each decision affects the outcome — win the treasure or lose the game.

---

## 🧠 Concepts Practiced

- Conditional statements (`if`, `elif`, `else`)
- Nested logic (decision trees)
- User input handling (`input()`)
- String normalization (`.strip()`, `.lower()`)
- Control flow
- Basic code structure using functions
