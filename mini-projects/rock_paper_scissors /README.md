# ✊✋✌️ Rock Paper Scissors (Python)

A simple command-line implementation of the classic Rock, Paper, Scissors game.  
Play against the computer and see if you can win!

---

## 🎯 Objective

The player selects Rock, Paper, or Scissors, and the computer randomly chooses one option.  
The winner is determined based on standard game rules.

---

## 🧠 Concepts Practiced

- Conditional statements (`if`, `elif`, `else`)
- Lists and indexing
- Random number generation (`random.randint`)
- User input handling (`input()`)
- Input validation
- Control flow
