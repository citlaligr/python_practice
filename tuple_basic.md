# 🧠 Exercise: Tuple Access

## 📝 Problem
Create a tuple named `person` that contains:
- a name (string)
- an age (integer)
- a height (float)

Using the values:
- "Rahul", 25, 5.9

Print the second element of the tuple.

## Expected Output 
Age: 25

---

## 💻 Solution
```python
person = ("Rahul", 25, 5.9)
print(f"Age: {person[1]}")
